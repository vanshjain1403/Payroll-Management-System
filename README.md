Payroll Management System

A simple Java-based Payroll Management System that helps manage employee records and calculate salaries for both full-time and part-time employees.
The project demonstrates key Object-Oriented Programming (OOP) concepts such as Abstraction, Inheritance, Encapsulation, and Polymorphism.

It uses a menu-driven console interface where users can add, remove, and view employee details dynamically.

Features

Add Full-Time Employees with monthly salaries

Add Part-Time Employees with hourly pay

Calculate salaries automatically based on employee type

Remove employees by their unique ID

Display all employee details

Simple text-based console interface
